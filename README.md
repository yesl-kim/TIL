## 작성규칙

- 폴더명, 파일명은 영문 CamelCase로 작성합니다.
- 매일 공부한 것을 기록하되, 날짜가 아닌 주제별로 분류합니다.  
  (나중에 제가 찾아보기 쉽기 위함입니다.)
- 찾아보기 쉽도록 최대한 자세하게 기록하되, 너무 많은 시간을 쏟지 않도록 합니다.
- 철저히 공부한 것을 정리하는 목적으로 기록되며, 출처는 항상 상단에 명시합니다.
- 더 많은 공유가 필요하다고 생각되는 글은 블로그에 올립니다.
